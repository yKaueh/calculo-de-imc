# Calculadora de IMC

Uma aplicação web simples para calcular o **Índice de Massa Corporal (IMC)** com interface responsiva e moderna.

## Funcionalidades

- Calcula o IMC a partir do peso (kg) e altura (m).  
- Mostra a classificação do IMC (magreza, normal, sobrepeso, obesidade).  
- Input com **validação simples** e aviso em caso de digitação incorreta.  
- Interface responsiva, funcionando em desktop e mobile.  
- Inputs destacam-se em **amarelo ao serem selecionados** para melhor usabilidade.

## Tecnologias utilizadas

- **HTML5** – estrutura semântica.  
- **CSS3** – estilos e responsividade.  
- **JavaScript** – cálculo do IMC e manipulação DOM.

## Como usar

1. Clone o repositório:  

   ```bash
   git clone <link-do-repositorio>
   
2. Abra index.html no navegador.

3. Insira seu peso e altura nos campos correspondentes.

4. Clique em Calcular para ver seu IMC.

## Observações

- O IMC é apenas uma **medida geral** e não substitui avaliação profissional de saúde.
- A calculadora é **simples e didática**, ideal para fins educacionais e portfólio.
